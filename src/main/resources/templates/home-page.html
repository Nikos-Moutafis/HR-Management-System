Welcome to this application

This application allows users to register in order to use it. It provides functionality for creating, updating or deleting employees, employers, and projects. You can associate an employee with an employer, assign a project to an employee, and vice versa.

Not only can you establish relationships between entities, but you can also manage them according to your needs, such as releasing an employee from a project or an employer. All these operations are available in a clear way.

Below are the relationships between entities

Entity Relations
Employer Has many Employees, Has many Projects
Employee Belongs to one Employer and can exist without Employer, Can be associated with many Projects
Project Belongs to one Employer and cannot exist without Employer, Can have many Employees
Note: An Employee and a Project must belong to the same Employer in order to be associated in the application.
